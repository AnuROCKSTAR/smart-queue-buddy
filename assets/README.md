# Assets Folder

This folder contains all static assets for Smart Queue Buddy.

## Structure

```
assets/
├── images/          # Images, screenshots, and brand assets
├── icons/           # Custom SVG icons
├── fonts/           # Custom font files
├── logo.svg         # Main brand logo
└── README.md        # This file
```

## Usage

- **Images**: Product images, screenshots, diagrams
- **Icons**: Custom icons not covered by Bootstrap Icons
- **Fonts**: Custom typography fonts (Inter, etc.)
- **Logo**: Official Smart Queue Buddy logo in SVG format
